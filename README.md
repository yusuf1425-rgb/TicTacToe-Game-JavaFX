# TicTacToe-Game-JavaFX
A GUI-based TicTacToe game using JavaFX, built as part of my learning journey through the Java Programming MOOC (University of Helsinki).
# JDK (JavaFX Included)
Use a JavaFX-enabled JDK for seamless setup. Azul Zulu FX provides a distribution with JavaFX bundled: https://www.azul.com/downloads/?version=java-1-lts&package=jdk-fx#zulu
